Vue动态组件的运用

发表于 2017-04-08 分类于 Vue

组件在之前已经介绍过了，今天分享一下动态组件，通过使用保留的 元素，动态地绑定到它的 is 特性，我们让多个组件可以使用同一个挂载点，并动态切换

动态组件 :is的用法

其次，使用 元素，动态地绑定到它的 is 特性，is中的值是组件的名称

动态组件的原理步骤

  1. 它会根据component 标签里 :is 来寻找Vue实例中 data的数据

  2. 这时com指向com1的组件,所以component的内容为com1组件的内容
