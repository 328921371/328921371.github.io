Vue插槽的使用

发表于 2017-04-03 | 分类于 Vue |

为了让组件可以组合，我们需要一种方式来混合父组件的内容与子组件自己的模板。这个过程被称为内容分发 。Vue.js 实现了一个内容分发 API，参照了当前 Web Components 规范草案，使用特殊的 元素作为原始内容的插槽。
