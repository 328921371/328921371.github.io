Vue-cli 4.0 打包配置不同环境

发表于 2019-12-20 分类于 vue-cli

在之前已经写过了vue 2.0的打包配置,今天写一下4.0的配置,略微有些区别

相比较于之前的版本,4.0的配置显得较为简单一些

首先在根目录下创建三个文件

.env .env.test .env.production

  • .env 本地环境

    NODE_ENV=development

  • .env.test 测试环境

    NODE_ENV=test

  • .env.production 生产环境

    NODE_ENV=production

配置好后,在api.js中就可以直接使用
